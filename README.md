# GeoLocationApi


simple API to get metadata of country from lat and long coordinates


## run with 

`cargo run ` or 
`cargo run --release`

# Test the API here: 

** [http://geoapi.soerensserver.xyz/getIsoCodes?lng=12&lat=12](http://geoapi.soerensserver.xyz/getIsoCodes?lng=12&lat=12) **
